## 윈도우 기준
---

1. 루비 설치 
   https://rubyinstaller.org/downloads/

   ```CMD
   ruby -v
   ```
2. Jekyll 설치
   CMD
   ```CMD
   gem install jekyll bundler
   ```

3. 폴더 이동 및 gem, bundle install
   ```CMD
   gem install bundler
   bundler install
   bundle install?
   ```

4. Jekyll Server
   ```CMD
   bundle exec jekyll serve
   ```
   ctrl+c 를 눌러 종료

5. 블로그 꾸민 후 Github Push
   ```Bash
   git add *
   git commit -m "설명"
   git push -u origin main
   ```

6. github.io 페이지 확인
